Here is a video that shows the most used features of repotool.

repotool is part of the DevMine source code analysis framework. It is a command line tool that aggregates source code repositories metadata (such as VCS type, commits and so on). It can also be used in addition to the result produced by srctool and output a combined JSON object.

The result may be used as input for the srcanlzr tool.
