Here is a video that present the tool crawld.

crawld is a data crawler and a source code repository fetcher. It is able to crawl metadata about source code repositories and developers from websites such as GitHub. All data is then stored into a database.

Along with the crawler, the fetcher can clone and update source code repositories fo further analysis through the devmine's tools.
