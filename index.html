DevMine

DevMine is about evaluating developers skills based on their open-source contributions.

To achieve this goal, developers metadata and source code analysis are performed, leveraging content from platforms such as GitHub.

Learn more

Crawld

crawld is a data crawler and source code repository fetcher. It is able to crawl metadata about source code repositories and developers from websites such as GitHub. All data is then stored into a database. Along with the metadata, it is able to clone repositories source code for further analysis.

Learn more »

Srctool

srtool is a command line tool to manage source code parsers. It is able to download parsers from a web server, install them and run them.In short, it is a manager for source code parsers.

Learn more »

Repotool

repotool is a command line tool that aggregates source code repositories metadata (such as VCS type, commits and so on) in order to add it to the JSON files produced by srctool.

Learn more »

Srcanlzr

srcanlzr is a source code analyzer. It analyzes a generic JSON representation of the source code, as generated by a source code language parser (only Go, Java and Ruby parsers exist for now).

Learn more »

Features

features computes features from source code data and developers metadata. A feature in the context of the DevMine project is something like the proficiency of a user in a certain programming language or paradigm, a developer reputation on GitHub and so on.

Learn more »

API-server

devmine is the API server. It provides a JSON RESTful API to provide information computed by various DevMine sub-projects. It is also able to answer queries to rank and filter developers according to some user queries. Say a user wants to find a good Go developer with some experience in Java and functional programming as well, devmine is able to provide an answer to that.

Learn more »

Devmine-web

devmine-web is a web front-end to the API server. More precisely, it allows users to actually query the API server using an intuitive, user-friendly interface.
