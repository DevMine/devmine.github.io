Here is a short video that shows srctool in action. This is the first one of a series of videos that demo the various DevMine tools.

I suggest you watch it full screen for a better experience.

srctool is a command line tool to manage source code parsers. As such, it is able to install, delete or update parsers but moreover, it can be used to parse projects sources by running the language parsers and merging their output into one JSON.

For more information about this tool, head to its dedicated documentation page.
